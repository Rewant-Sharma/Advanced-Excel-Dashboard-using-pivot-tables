# Advanced-Excel-Dashboard-using-pivot-tables

Analyzed a transportation and supply-delivery dataset using advanced Excel (Pivot Tables, Slicers, Dashboards) to uncover category-wise spending patterns, monthly demand fluctuations, and item-level purchase behavior. Built an interactive dashboard that drove insights for transportation capacity planning, vendor optimization, and cost-effective procurement decision-making.

• Built an interactive Excel dashboard by integrating Pivot Tables and dynamic charts, enabling instant visibility into category spend, volume trends, and monthly fluctuations that improved operational decision-making for transportation and procurement.
• Identified cost concentrations across Accessories, Clothing, and Footwear by analyzing purchase amounts and SKU-level patterns which helped optimize vendor negotiations and inventory planning.
• Uncovered monthly demand spikes through trend analysis of purchase values and item counts resulting in more accurate forecasting and proactive transportation scheduling.
• Enhanced route and load planning by mapping high-volume categories and sub-categories leading to better fleet utilization and reduced shipment bottlenecks.
• Improved procurement efficiency by transforming raw operational data into a structured, insight-driven dashboard that enabled data-backed decisions for cost control and supply readiness.
